# Beyond Vanilla

**Beyond Vanilla** is a Minecraft Forge mod focused on expanding the vanilla experience without losing its original feel.

Instead of completely changing the game, the goal is to **build on top of vanilla mechanics**, adding new RPG-like features, items, and progression systems while keeping everything familiar and immersive.

---

## ✨ Features

* New items and equipment
* RPG-inspired progression systems
* Vanilla-style balancing
* Seamless integration with existing gameplay
* Designed to feel like a natural extension of Minecraft

> The mod aims to answer: *"What if vanilla had just a bit more depth?"*

---

## 🎯 Design Philosophy

Beyond Vanilla follows a few core principles:

* **Vanilla First** – Nothing should feel out of place
* **Minimal Intrusion** – No unnecessary complexity
* **Progression Matters** – Players should feel growth over time
* **Consistency** – Everything should match Minecraft’s style

---

## ⚙️ Requirements

* Minecraft Forge (1.12.2)
* Java 8+

---

## 📦 Installation

1. Install Minecraft Forge for 1.12.2
2. Download the latest version of the mod
3. Place the `.jar` file into your `mods` folder
4. Launch the game

---

## 🛠️ Development

This project uses Gradle.

To build:

```
./gradlew build
```

The compiled `.jar` will be located in:

```
build/libs/
```
